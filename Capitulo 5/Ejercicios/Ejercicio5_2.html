5.2 Ejercicio de autoevaluación | Indique si cada uno de los siguientes enunciados verdadero o falso. Si es falso, explique por qué:

  • a) La regla @font-face especifica que se utilizará una fuente incrustrada cuando el documento se despliegue en una pantalla de computadora.

    • >> Falso, @media screen especifica que se usa una fuente incrustrada cuando el documento se muestra en la pantalla.

  • b) Podemos usar gradiantes en cualquier propiedad que acepte una imagen.

    • >> Verdadero.

  • c) Un gradiante horizontal cambia en forma gradual de arriba hacia abajo.

    • >> Falso, los gradiantes horizontales cambian en forma gradual de izquierda a derecha.

  • d) Podemos agregarlíneas entre columnas mediante el uso de la propiedad column-gap.

    • >> Falso, se pueden agregar líneas entre columnas haciendo uso de column-rule.

  • e) La regla @media determina el tipo y tamaño del dispositivo en el que se despliega la página. Cuando el navegador examina la regla, el resultado es verdadero o falso. Los estilos de la regla se aplican sólo si el resultado es falso.

    • >> Los estilos de la regla @media se aplican sólo si el resultado es verdadero.

  • f) Para agregar múltiples imágenes de fondo a un elemento, usamos la propiedad background-position para especificar en dónde colocar cada imagen mediante el uso del modelo de cajas.

    • >> Falso, background-origin especifica en dónde se va a colocar cada imagen mediante el uso del modelo de cajas.
